1. What is a Table?

Table is the most important object of a database. All data in a relational database is stored in a table, which is made up of columns (fields) and rows (records). Table is also known as relation.

2. List 3 methods for creating tables in MS Access.

The three methods for creating for creating tables in MS Access are:
  1. Create table in Design view.
  2. Create table by using wizard.
  3. Create table by entering data.

3. Define TEXT data type in MS Access.

TEXT data type is used to store alphanumeric (alphabets and numbers) and special characters. It can store up to 255 characters. The default length of TEXT data type is 50 characters.

4. What is the Auto-Number data type in MS Access?

Auto-Number data type is used to generate the next number automatically. When a new record is to be inserted, the Auto-Number field's value will be the incremented value of last record's Auto-Number value.

5. List different data types in MS Access.

MS Access provides the following data types:
  • Text
  • Memo
  • Number
  • Data/Time
  • Currency
  • Auto Number
  • Yes/No
  • OLE object
  • Hyperlink

6. What is meant by degree of a relation?

The number of fields in a relation or table is called degree of a relation. For example, a table with 10 fields has a degree of 10. Once a table is created, its degree usually does not change. However, the degree of relation remains constant.

7. Define field property.

Field properties are used to define how data will be inserted, stored and displayed in MS Access. The properties of each field can be set in design view.

8. Write two field types used in MS Access.

Two of the most commonly used field types are:
  • Text
  • Number

9. Define Query.

A query is a statement that extracts specific information from database. Queries are used to retrieve the required data from database. Data is retrieved according to the criteria given by the user. It is a more flexible way of selecting, filtering and sorting records.

10. What is Action query?

An action query is used to make changes in specified records in an existing table. It can also be used to create a new table. There are four types of action queries: Create Table, Append, Update and Delete queries.

11. What is sorting?

The process of arranging data or records in a sequence is called sorting. Sorting can be done in ascending order or descending order.

12. State the use of wildcard.

Wildcards are the special characters that are used in queries to specify the criteria based on a pattern. The two wildcards and their uses are:
  • *: Specifies any number of characters. For example, AC* specifies a text which starts with AC, followed by anything of any length.
  • ?: Specifies a single character. For example, AC? specifies a text which starts with AC, followed by only one character.

13. What is a filter?

A filter is used to extract the records that match a set of criteria. Filters are basically queries but they only appear to open tables or forms.

14. What is the use of datasheet view?

The table view that is used to enter, delete or modify data in a table is called datasheet view. The table in this view is displayed in rows and columns. The name of each field is displayed at the top of the column as header. Each row contains a complete record.

15. Name two table views available in MS Access?

There are two table views in MS Access:
  • Datasheet view
  • Design view

16. List the advantages of Query.

Some advantages of using query are flexibility, ability to write automated programs for managing database, facility of joining, ease of use.

17. What is JOIN?

A query that extracts data from multiple related tables is called JOIN. A JOIN uses the relationship of the tables for extracting data from the tables. A JOIN reaches the required records in the first table and then reaches the corresponding record in the second tables.
