1. What is a Report?

Reports are the output of a database application. The user can generate different types of reports by manipulating the database. The information on the reports is arranged in different styles. The reports may contain graphs and charts.

2. What is the importance and advantages of Report?

Reports are important because they provide the following advantages:
  • Reports present the required information in formatted style.
  • Reports provide flexibility to present the same data in different styles.
  • Reports can display information with graphics and charts.

3. Write the uses of reports.

Reports are used for the following reasons:
  • To produce output in a formatted way according to the needs of the user.
  • To display the data retrieved from different tables.
  • To display the result of queries.

4. What is a Form?

The form object of database is most commonly used for manipulating tables using a graphical user interfaces. A form can be used to display data, and manipulate tables at a time. Similarly, forms allow the user to enter and modify data in multiple tables at a time.

5. Write two advantages of Form.

Some advantages of form are:
  • Easier to use and user-friendly.
  • No technical knowledge required to manipulate tables.
  • Time saving and flexibility.

6. Define Subform.

A form that is placed in a parent form or main form is called the subform. Subform is also known as child form. A subform is particularly used when tables are associated with each other.

7. What is Auto-Form?

MS Access also provides a method to create a form automatically based on the table or query you select. Auto-Form does not ask to specify fields and other options as form wizard does. It creates a form very quickly by including all the fields of the selected table or query.

8. What is a Columnar Form?

Columnar form is used to display one record at a time. It displays textboxes and labels. The textboxes represent the fields of the table or query. The labels represent the names of the fields.

9. Write the uses of Tabular Form.

Tabular form is used to display many records at a time. It displays them as a table. Each row in this form displays one record from the table.

10. Which two database objects can be used to create a report?

The following two objects can be used to create a report in MS Access:
  • Tables
  • Queries

11. Define List box.

List is used to display a list of items in forms. The user can select the desired item from the available items in the list. The user can select one or more items. List box occupies a specified space on the form. A scrollbar appear in the list box if the number of items are more than the available space.

12. How is combo box used?

Combo box is used to display a list of items in forms. It consists of a textbox and a dropdown list. The user can select the desired item from the combo box. Combo box contains an arrow on the right side. The user can click on the arrow to see the list of all available items.

13. What is a Radio button?

The radio button control is used to display a set of choices or options from which the user can select only one. This button appears on the form as a small circle. When it is selected, a dot appears in the circle (the circle is filled). If another option is selected afterwards, the previously selected radio button gets de-selected automatically, and the newly selected button gets filled.

14. What are checkboxes?

Checkboxes are used to display Yes/No, True/False or On/Off option. It is used when the user can select one or more options at the same time. The user can select any or all values from a group of checkboxes. A checkbox can be selected by clicking on it.

15. Differentiate between Form and Query.

Forms are used to enter data in tables easily using graphical user interface. Forms consist of buttons, textboxes and lists.

Query is used to retrieve data from tables. It requires a certain level of technical knowledge.

16. Differentiate between Form and Report.

The main difference between form and report is that, form is used to enter data into tables. While reports are used to present the stored data in a formatted way.
