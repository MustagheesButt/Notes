1. Define an IDE.

IDE stands for Integrated Development Environment. It is a collection of facilities provided to the users. It is used to create databases and database applications. An IDE simplifies the tasks of creating and using a database.

2. List some advantages of an IDE.

Some advantages of an IDE are:
  • It is easy to use.
  • It simplifies the creation of database applications.
  • It does not require a detailed knowledge of database.
  • It provides predefined facilities to search, sort, retrieve and process data.

3. Define RDBMS.

RDBMS stands for Relational Database Management System. It is a collection of programs that are used to manage relational databases. A relational database system provides various operations used to manipulate the data in the database tables. In relational database system, a table is referred to as relation. Microsoft Access is an example of a relational database management system.

4. Define database objects.

A component of a database system is known as a database object. MS Access provides various objects like Tables, Queries, Forms and Reports that are used to store and retrieve data from database.

5. Write down any two database objects.

Some database objects are:
  • Tables.
  • Forms.
  • Queries.
  • Macros.

6. What is a database wizard?

A database wizard is a set of steps that guides the user to create and setup a database easily. It includes choosing a database template, selecting fields, making customizations, adding pictures in the database.

7. List some facilities provided by a RDBMS.

Some facilities provided by a good modern RDBMS are:
  • Data Security.
  • Data Integrity.
  • Easy Access and Simple Interface.
  • Backup and Recovery.

8. List the major inputs to a database design.

The major inputs for database design are:
  • Table.
  • Form.
  • Query.
  • Report.

9. What is MS-Access? List its uses.

Microsoft Access is one of the most popular and powerful DBMS. It provides various built-in features. These features help the user to construct a database, manipulate it, and view information easily a graphical interface.

10. Describe the menu bar.

The commonly used commands are grouped together on the menu bar for quick access. These commands can be used to perform different tasks on the database. Each menu contains related commands to perform different operations on the database.

11. What is an OLE object in MS-Access?

OLE stands for Object Linking and Embedding. This data type is used to enter objects from other applications such as a spreadsheet created in Excel or picture file.

MS Access provides the following data types:

  • Data/Time
  • Currency
  • Auto Number
  • Yes/No
  • OLE object
  • Hyperlink

14. What is the importance of Report?

Reports are very important in making decisions. The user can send reports to different people. The output of a query can also be used as source for reports. Most of the DBMSs provide the facility of reports.

15. What is Query?

A table can be large depending on the information it holds. To quickly access the data you want, queries are used. In order to limit part of the data in a table for a specific goal, for better management and quick searching, queries can be used by specifying exactly what kind of data you want to extract from a table. They can also be used for updating rows, and performing other operations on a table.

16. How is a Query written?

A query is written in database access language such a SQL. A query retrieves data from a table in the database and displays it in a formatted and easy to understand way. The user can also change data in the database by using queries. Queries can also be used to perform calculations on fields.

17. List the important purposes of Query language.

Some important purposes of a query language are:
  • Create table structures.
  • Enter data in tables.
  • Retrieve data from tables.
  • Update data in tables.
  • Delete data from tables.
